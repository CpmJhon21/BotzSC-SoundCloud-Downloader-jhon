POST /ai/gpt4

GPT-4

Access GPT-4 AI model for chat completion

Parameters

Body Parameters (JSON)

Parameter Type Required Description
messages array Yes Body parameter messages
user_id string Yes Body parameter user_id

Code Examples

cURL
bash 
curl -X POST "https://host.optikl.ink/ai/gpt4" \
  -H "Content-Type: application/json" \
  -d '{
  "messages": [
    {
      "role": "system",
      "content": "Good Person"
    },
    {
      "role": "user",
      "content": "hai"
    }
  ],
  "user_id": "1234"
}'
JavaScript
javascript 
fetch('https://host.optikl.ink/ai/gpt4', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
  "messages": [
    {
      "role": "system",
      "content": "Good Person"
    },
    {
      "role": "user",
      "content": "hai"
    }
  ],
  "user_id": "1234"
})
})
.then(response => response.json())
.then(data => console.log(data))
